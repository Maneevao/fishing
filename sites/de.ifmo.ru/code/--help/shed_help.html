Руководство по работе с системой формирования расписания
График аттестации
Аттестация проводится в соответствии с графиком, формирующимся на основании заявок учебных кафедр. График публикуется на сайте СДО (http://cde.ifmo.ru) в разделе "Расписание аттестаций", подразделы "осенний семестр" или "весенний семестр".


На странице соответствующей группы приводится список дисциплин, по которым необходимо пройти тестирование. Справа от названия темы аттестаций по дисциплине указан крайний срок сдачи соответствующего теста. Если номер группы не является ссылкой, то для этой группы аттестации в СДО в текущем семестре не предусмотрены.


Для прохождения аттестации в ауд.101 студенту необходимо лично подать заявку через сайт системы ДО (раздел "Расписание аттестаций") на удобные день и время.

Подача заявок
Для того чтобы подать заявку, в разделе "Расписание аттестаций" необходимо выбрать семестр и неделю, когда планируется работа в компьютерном классе (ауд.101).


В появившемся расписании отображается информация о количестве свободных мест в компьютерном классе (ауд.101) и поданных заявок от студентов на соответствующие сеансы по дням недели. Элементы столбца "Время" являются ссылками на окно входа в систему автоматического формирования расписания, в котором требуется ввести параметры своей учетной записи (логин, пароль и роль).


В случае успешной проверки параметров учетной записи появится окно "Расписание аттестаций студента", где отображаются данные студента (фамилия, имя, отчество, № группы), месяц, год, список заявок, установленный лимит времени, остаток времени (в сеансах по 40 мин.) и штраф, который начисляется вам за неявку на заявленный сеанс.
Лимит времени работы в компьютерном классе ЦДО (ауд.101) устанавливается на месяц. Величина лимита определяется на основании графика аттестаций для каждого студента, исходя из количества тем, по которым необходимо пройти аттестацию, времени тестирования по каждой теме и объему обучающихся ресурсов системы по дисциплинам, включенным в график аттестаций.
Остаток времени - это время оставшиеся после выполнения или невыполнения заявок. Если заявка "не выполнена", то из лимита времени работы в компьютерном классе ЦДО (ауд.101) вычитается не только заявленное время сеанса, но и штраф.
Штраф - рассчитывается по формуле: S=k*T, где k - коэффициент, устанавливаемый ЦДО на месяц (обычно k=2), T - продолжительность сеанса.
Список заявок на выбранный месяц содержит информацию о дате, времени начала и окончания сеанса, дате подачи заявки и ее состоянии. Предусмотрены следующие состояния: "рассматривается", "удовлетворена", "отклонена", "выполнена", "не выполнена", "снята".
Если заявка подана на следующий месяц, то для того чтобы поданная заявка отобразилась в списке заявок необходимо в соответствующем поле поменять месяц.


Заявка на аттестацию может быть подана, если лимит времени, предоставленный администратором на месяц, не исчерпан, и нет заявок в состоянии "рассматривается" на другие дни.
Для подачи заявки необходимо выбрать ссылку "Подать заявку".


В появившемся окне приведена информация о правилах работы в компьютерном классе (а.101), поля "Дата", "Начало", "Окончание". Для подачи заявки необходимо выбрать из списка дату, время начала сеанса, время окончания сеанса, при этом в окне отобразится количество свободных мест на выбранный вами сеанс, и перейти по ссылке "Сохранить". Если количество свободных мест: 0, ссылка "Сохранить" будет недоступна.
Если уже существует заявка в состоянии "рассматривается", поданная на другой день, то система выдаст сообщение "Заявка не принята! В системе запрещено подавать заявки на разные дни".


Если заявляемое время превышает остаток времени, то система выдаст сообщение "У вас недостаточно времени чтобы подать эту заявку".
Если подаваемая заявка приведет к образованию сеанса, время которого превышает заданную продолжительность непрерывной работы, то система выдаст сообщение "С учетом ранее поданных заявок на [выбранная дата] продолжительность непрерывной работы в классе превышает установленное ограничение - 3 ч.".
Если подаваемая заявка соответствует всем критериям, появится окно с сообщением, что "заявка принята к рассмотрению". Далее по ссылке "Вернуться назад" необходимо перейти в окно, где отображается список заявок. Обязательно проверьте правильность параметров поданной заявки.
Вновь поданная заявка будет иметь состояние "рассматривается". Поданные заявки рассматриваются накануне тестирования. Если заявка принята, то она принимает состояние "удовлетворена", в противном случае - "отклонена".
Отзыв заявки
Заявка, имеющая состояние "рассматривается", может быть отозвана студентом самостоятельно по ссылке "рассматривается". При этом заявленное время сеанса автоматически возвращается студенту в полном объеме.
В открывшемся окне "Отозвать заявку" необходимо щелкнуть по ссылке "Отозвать", после чего появится окно с информацией о том, что заявка отозвана.


По ссылке "Вернуться назад" необходимо перейти в окно, где отображается список заявок.
Снятие заявки
Если заявка принята и имеет состояние "удовлетворена", а у студента нет возможности ее выполнить, необходимо снять заявку до момента начала ее действия. При этом заявленное время сеанса все равно будет вычтено из лимита, но не будет начислен штраф.
Ссылка "удовлетворена" предназначена для перехода в окно "Снять заявку", где необходимо щелкнуть по ссылке "Снять".


Появится окно с предупреждением "Внимание! При снятии заявки штраф не накладывается, но заявленное время вычитается из лимита".
Для подтверждения отзыва заявки необходимо щелкнуть по ссылке "Продолжить". Если вы не хотите снимать заявку - необходимо выбрать ссылку "Вернуться назад", и заявка останется в силе.
Информация о причине отказа
Если заявка имеет состояние "отклонена", то причину отказа можно узнать, перейдя по ссылке "отклонена" в окне "Причина отказа".


По ссылке "Вернуться назад" выполняется переход в свое окно "Расписание аттестаций студента".
Сообщения об ошибке
По ссылке "Сообщения об ошибке" вы можете сообщить об опечатке или неточности в тестовом задании.


В окне "Список сообщений об ошибках" выберите ссылку "Сообщить об ошибке". В появившемся окне введите требуемый текст и после этого, выберите ссылку "Сохранить".


В появившемся окне появится информация, что сообщение об ошибке принято. По ссылке "Вернуться назад" выполняется переход в свое окно "Расписание аттестаций студента".
