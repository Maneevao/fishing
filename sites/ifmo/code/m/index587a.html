  • Приложение для работы с ЭИОС Университета ИТМО

    Мобильное приложение (OS Android) для работы с электронной информационно-образовательной средой Университета ИТМО . Разработчик: Максим Рудик.
    Возможности: просмотр электронного журнала за весь период обучения, просмотр подробного отчёта об успеваемости по дисциплине, просмотр протокола изменений, рейтинга, расписания занятий с автоматическим выбором текущей недели, просмотр на карте города корпуса университета, в котором будут проходить занятия, просмотр расписания сессии, расписания преподавателя, включая поиск по преподавателям, просмотр расписания аттестаций с автоматическим выбором ближайшей к сдаче недели, а также возможность записи на сеанс тестирования и уведомления о новых баллах.

  • Расписание Университета ИТМО (приложение для Android)

    Приложение для просмотра расписания занятий Университета ИТМО. Разработчик Alexey Pavlenko.
    Позволяет просматривать расписание занятий группы, преподавателя, включая поиск по преподавателю. Обеспечивается определение чётности и номера недели, автоматический показ текущей недели; отображение места, времени, названия пары, преподавателя, номера группы в расписании преподавателя. Расписание сохраняется для дальнейшего просмотра в режиме оффлайн.

  • Интерактивная система по векторной алгебре и аналитической геометрии

    Интерактивная система предназначена для работы студентов с учебными материалами по векторной алгебре и аналитической геометрии как во время аудиторных занятий, так и самостоятельно.
